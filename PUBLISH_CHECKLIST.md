# Publish checklist

## Upload these repository files

Upload `README.md`, `CHANGELOG.md`, `LICENSE`, `docs/`, and `.github/` to the repository.

## Create the v1.0.0 release

1. Open the repository.
2. Click **Releases** on the right.
3. Click **Create a new release**.
4. Tag: `v1.0.0`
5. Release title: `ShadowRotation v1.0.0`
6. Paste `release/RELEASE_NOTES.md`.
7. Attach `release/ShadowRotation_v1.0.0_SHAREABLE_INSTALL.zip`.
8. Click **Publish release**.

## Enable GitHub Pages

1. Open repository **Settings**.
2. Open **Pages**.
3. Set Source to **Deploy from a branch**.
4. Choose branch `main` and folder `/docs`.
5. Save.

Your site will be:

https://turtlewownub.github.io/ShadowRotation/
