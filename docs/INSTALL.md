# Installation

1. Close Turtle WoW.
2. Download `ShadowRotation_v1.0.0_SHAREABLE_INSTALL.zip` from Releases.
3. Extract the included `ShadowRotation` folder into:

```text
Turtle WoW/Interface/AddOns/
```

4. Verify this exact path exists:

```text
Turtle WoW/Interface/AddOns/ShadowRotation/ShadowRotation.toc
```

5. Start Turtle WoW and run:

```text
/shadow version
/shadow modules
```

The module check should end with `System Status: READY`.

Recommended macro:

```text
/script ShadowRotation()
```
