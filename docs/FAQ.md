# FAQ

## The launcher says “Missing .toc file”

The final path must be:

```text
Turtle WoW/Interface/AddOns/ShadowRotation/ShadowRotation.toc
```

## Trackers are missing

```text
/shadow recoverui
/shadow tracker on
/shadow dots on
/shadow cds on
/shadow queue on
```

## Minimap button is missing

```text
/shadow minimap on
```

## Reporting a bug

Include the exact error, `/shadow version`, `/shadow modules`, and what you were doing when it happened.
