Installation

No installer program is needed. Download the ZIP and copy one folder.

Download the latest ZIP from GitHub Releases.
Close Turtle WoW.
Extract the included ShadowRotation folder.
Move it into Turtle WoW/Interface/AddOns/.
Confirm ShadowRotation.toc is directly inside the addon folder.

Recommended macro

/script ShadowRotation()

First-run check

/shadow version
/shadow modules

The module check should end with System Status: READY.

Frequently asked questions

The launcher says “Missing .toc file.”

Make sure the final path is Interface/AddOns/ShadowRotation/ShadowRotation.toc. The file must be directly inside the ShadowRotation folder.

My trackers are missing.

Run /shadow recoverui, followed by /shadow tracker on.

The minimap button is missing.

Run /shadow minimap on.

How do I report a problem?

Open a GitHub Issue and include the exact error, /shadow version, and /shadow modules output.
