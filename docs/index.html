Installation

1. Download the latest ZIP from Releases.
2. Close Turtle WoW and extract the included ShadowRotation folder.
3. Move it into Turtle WoW/Interface/AddOns/.
4. Confirm Interface/AddOns/ShadowRotation/ShadowRotation.toc exists.
5. Start the game and run:
/shadow version
/shadow modules

Recommended macro

/script ShadowRotation()
