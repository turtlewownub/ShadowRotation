Why ShadowRotation?

ShadowRotation is more than a spell recommendation addon. It explains decisions, tracks recent fights, supports multiple gameplay profiles, and helps Shadow Priests improve without filling the screen with unnecessary UI.

Installation

Download the latest ZIP from the Releases page.
Close Turtle WoW and extract the included ShadowRotation folder.
Move it into Turtle WoW/Interface/AddOns/.
Confirm Interface/AddOns/ShadowRotation/ShadowRotation.toc exists.
Start the game and run the checks below.
/shadow version
/shadow modules

The module check should end with System Status: READY.

Recommended macro

/script ShadowRotation()

Quick commands

/shadow options
/shadow profile raid
/shadow pack standard
/shadow coach
/shadow insights
/shadow recoverui
