INSTALLATION

Normal install:
1. Close Turtle WoW.
2. Open the game's Interface\AddOns folder.
3. Remove any existing ShadowRotation folder.
4. Extract the ShadowRotation folder from the install ZIP into AddOns.
5. Verify this exact path exists:
   Interface\AddOns\ShadowRotation\ShadowRotation.toc
6. Start the game.

Do not place README, scripts, or a source-project folder directly in AddOns.

Update install:
Replacing the addon folder normally preserves settings because settings live under the WTF folder.

Recommended first checks:
/shadow version
/shadow modules
